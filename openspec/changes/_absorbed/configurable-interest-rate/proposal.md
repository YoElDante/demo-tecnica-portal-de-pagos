# Proposal - Configurable Interest Rate

## Problema

La tasa de interes aparece en la documentacion como configurable, pero el sistema aun depende de un valor fijo en partes del codigo.

## Objetivo

Mover la configuracion de tasa a una fuente parametrizable por municipio o entorno.

## Alcance

- Ajuste en capa de configuracion
- Consumo desde servicios de deuda
- Documentacion y ejemplos de entorno
